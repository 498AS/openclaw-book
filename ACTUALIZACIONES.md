## Febrero 2026

### Anthropic lanza Opus 4.6 con Agent Teams
**5 febrero 2026**

Anthropic lanzó Claude Opus 4.6 con una nueva funcionalidad revolucionaria: **Agent Teams**. Permite orquestar 20-30 agentes trabajando en paralelo, coordinándose autónomamente.

**Características:**
- Un agente actúa como "team lead" coordinando el trabajo
- Los agentes se comunican directamente entre sí
- Cada agente tiene su propia ventana de contexto
- Se activa con `CLAUDE_CODE_EXPERIMENTAL_AGENT_TEAMS=1`

**Caso de uso destacado:** Un equipo usó 16 agentes para escribir un compilador de C en Rust (100.000 líneas) capaz de compilar el kernel de Linux.

**Fuentes:**
- [Anthropic - Introducing Claude Opus 4.6](https://www.anthropic.com/news/claude-opus-4-6)
- [TechCrunch - Anthropic releases Opus 4.6](https://techcrunch.com/2026/02/05/anthropic-releases-opus-4-6-with-new-agent-teams/)

### GitHub lanza Agent HQ
**Febrero 2026**

GitHub ha lanzado **Agent HQ**, una plataforma que integra múltiples agentes de IA:

- Codex (OpenAI)

Disponible para suscriptores Pro+ y Enterprise.

**Fuente:** [GitHub Blog - Agent HQ](https://github.blog/news-insights/company-news/pick-your-agent-use-claude-and-codex-on-agent-hq/)

### Apple integra agentes en Xcode 26.3
**Febrero 2026**

Xcode 26.3 integra Claude Agent y OpenAI Codex directamente en el entorno de desarrollo de Apple. Los agentes pueden:
- Escribir código autónomamente
- Compilar proyectos
- Ejecutar tests
- Verificar visualmente su propio trabajo

**Fuente:** [VentureBeat - Apple Xcode integration](https://venturebeat.com/technology/apple-integrates-anthropics-claude-and-openais-codex-into-xcode-26-3-in-push)

### MOLTbook: Red social construida por un agente
**Enero 2026**

Un agente de OpenClaw llamado "Clawd Clawderberg" (creado por Matt Schlicht) construyó **MOLTbook**, una red social exclusiva para agentes IA. Los agentes:
- Publican contenido
- Comentan y discuten
- Votan entre ellos
- Forman "opiniones" colectivas

Los humanos solo pueden observar, no participar.

**Fuente:** [TechCrunch - OpenClaw AI social network](https://techcrunch.com/2026/01/30/openclaws-ai-assistants-are-now-building-their-own-social-network/)

### Cursor construye navegador con swarm de agentes
**23 enero 2026**

Cursor (editor de código con IA) demostró un experimento donde cientos de agentes trabajaron durante una semana sin intervención humana, construyendo un navegador web funcional.

El sistema incluía:
- Agentes "planificadores"
- Agentes "trabajadores"
- Agentes "jueces"

Coordinándose sobre millones de líneas de código.

**Fuente:** [Fortune - Cursor agent swarm](https://fortune.com/2026/01/23/cursor-built-web-browser-with-swarm-ai-agents-powered-openai/)
