# OpenClaw Book — Código y Ejemplos

> Repositorio companion del libro **"OpenClaw: Guía práctica para construir y mantener a raya agentes de IA que trabajan para ti las 24 horas"**

Por Carlos Ortet, con la imprescindible colaboración de Tomás Awate y el intrépido equipo de 498AS.

---

## Qué encontrarás aquí

Todo el código del libro, organizado por capítulos. Si algo en las páginas impresas no funciona, mira aquí: lo mantenemos actualizado.

### Estructura

```
openclaw-book/
├── capitulo-05/          # Configuración esencial
│   ├── settings.json
│   └── allowedTools.json
├── capitulo-06/          # Integraciones
│   ├── whatsapp-basic/
│   ├── telegram-setup/
│   └── slack-webhook/
├── capitulo-07/          # Hacks y trucos
│   ├── hooks/
│   ├── skills/
│   └── automation-scripts/
├── capitulo-08/          # Seguridad
│   └── security-configs/
├── capitulo-09/          # Multi-agente
│   └── multi-agent/
├── capitulo-11/          # Monitorización
│   └── monitoring/
├── anexos/
│   ├── anexo-a-instalacion/
│   ├── anexo-b-comandos/
│   └── anexo-c-troubleshooting/
└── ejemplos-extra/
    └── recetas/          # Las 10 recetas de la contraportada
```

## Cómo usar este repositorio

1. **Clona el repo:**
   ```bash
   git clone https://github.com/498as/openclaw-book.git
   ```

2. **Navega al capítulo que te interesa**

3. **Copia y adapta** el código a tu configuración

## Sobre OpenClaw

OpenClaw (anteriormente moltbot, anteriormente clawdbot) fue creado por **Peter Steinberger**, fundador de PSPDFKit. Peter liberó el proyecto como open source, haciendo posible todo lo que ves aquí.

- **Repositorio oficial:** github.com/steipete/agent-rules
- **Documentación:** docs.openclaw.ai
